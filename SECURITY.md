# Security Policy

## Supported Versions

Only the latest release of DevAudit is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in DevAudit itself, please do **not** open a public issue.



We will acknowledge receipt of your report within 72 hours, coordinate a fix, and publish a security advisory once resolved. We appreciate your efforts in keeping DevAudit secure for everyone.
